This is just a small Brainfuck interpreter written in Python.

Usage:
```batch
python brainfuck.py yourcode.bf
```

You can use it as a module as well:
```python
import brainfuck

sourcecode = """
++++++++++[>+++++++>++++++++++>+++>+<<<<-]
>++.>+.+++++++..+++.>++.<<+++++++++++++++.
>.+++.------.--------.>+.>.
"""

brainfuck.evaluate(sourcecode)
```
